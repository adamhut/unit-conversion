## Installation

You can install the package via composer:

```bash
composer require adamhut/unit-conversion
```

## Usage

``` php
$skeleton = new Adamhut\UnitConversion();
echo $skeleton->echoPhrase('Hello, Adamhut!');
```

## Testing

``` bash
composer test
```
